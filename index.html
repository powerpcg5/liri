Liri – A Node.js command-line utility for querying Bands in Town, Spotify, and OMDB APIs

Synopsis
            node liri concert-this <artist or band name>
            node liri spotify-this-song <song name>
            node liri movie-this <movie name>
            node liri do-what-it-says <command file>

Commands may be abbreviated.  The following commands are supported:

  • concert-this:  Query the Bands in Town API for an artist’s or band’s upcoming concerts, including information on concert dates, venues, and venue locations;
  • spotify-this-song:  Query the Spotify API for a song’s artist(s), album, and preview URL on Spotify;
  • movie-this:  Query the OMDb API for a movie’s year of release, IMDb rating, Rotten Tomatoes rating, country of origin, language(s), plot, and actor(s);
  • do-what-it-says:  Run an external file (random.txt by default) containing any of the above commands.
Installation

To install and set up Liri:

  1. In a Terminal (or equivalent) window, clone the Liri GitHub repository with:
    git clone https://github.com/powerpcg5/liri.git
  2. cd liri;
  3. vi .env, and append the following lines to the file:
    # Spotify API keys
    SPOTIFY_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
    SPOTIFY_SECRET=yyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyy
    (Replace xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx and yyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyy with your Spotify client ID and secret, respectively, which you can obtain from here);
  4. npm init -y
    (This is actually optional, as package.json is included); and finally,
  5. npm i
